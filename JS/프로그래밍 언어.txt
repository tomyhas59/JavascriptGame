프로그래밍 언어 (Programming Languages):
JavaScript: 웹 개발에서 가장 널리 사용되는 스크립트 언어로, 프론트엔드 및 백엔드 개발에 모두 사용됩니다.

Python: 다목적 프로그래밍 언어로, 웹 개발, 데이터 과학, 인공 지능, 자동화 등 다양한 분야에서 사용됩니다.

Java: 크로스 플랫폼 언어로, 백엔드 서버 개발, 안드로이드 앱 개발에 널리 사용됩니다.

Ruby: 직관적이고 간결한 문법을 가진 언어로, 웹 개발 프레임워크인 Ruby on Rails와 함께 주로 사용됩니다.

C#: Microsoft의 개발 플랫폼에 사용되며, 게임 개발, Windows 애플리케이션, 웹 애플리케이션 등에서 사용됩니다.

프론트엔드 프레임워크 (Frontend Frameworks):
React: Facebook에서 개발한 라이브러리로, 사용자 인터페이스 개발에 주로 사용되며, 컴포넌트 기반 아키텍처를 제공합니다.

Angular: Google에서 개발한 프론트엔드 프레임워크로, 기능이 풍부하고 큰 규모의 애플리케이션을 개발하는 데 적합합니다.

Vue.js: 간단하고 가벼운 프론트엔드 프레임워크로, 쉬운 학습 곡선과 유연한 사용성을 제공합니다.

백엔드 프레임워크 (Backend Frameworks):
Express.js: Node.js를 위한 가장 인기 있는 백엔드 프레임워크로, 경량이면서 유연한 웹 애플리케이션 개발을 가능하게 합니다.

Django: Python 기반의 고수준 웹 프레임워크로, 안정성과 생산성을 중시하는 웹 애플리케이션을 빠르게 개발할 수 있습니다.

Ruby on Rails: Ruby 언어를 기반으로 하는 백엔드 웹 개발 프레임워크로, 개발자 친화적이고 생산성이 뛰어납니다.

Spring Boot: Java 기반의 백엔드 프레임워크로, 엔터프라이즈급 애플리케이션 개발에 적합하며, Spring 프레임워크의 특징을 활용합니다.

Ruby on Rails: Ruby 언어를 기반으로 하는 백엔드 웹 개발 프레임워크로, 개발자 친화적이고 생산성이 뛰어납니다.
